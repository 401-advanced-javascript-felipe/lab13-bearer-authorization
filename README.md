### Setup
#### `.env` requirements
* `PORT` - Port Number
* `MONGODB_URI` - URL to the running mongo instance/db
* `SECRET` - a secret for jwt
* `TIMEOUT` - a testing timeout

#### Running the app
* `npm start`
